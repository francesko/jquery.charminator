jQuery.charminator

A jQuery plugin to do crazy stuff with text! Try it out in the test below or visit the github page for more info. If you are too lazy to click a link, download the source directly here or get the minified version here.

Play with the parameters



Note that this plugin works better with monospaced font families as the size of all letters are the same (this avoids flickering of the text when changing quickly from one letter to another).

DISCLAIMER: As of version 0.0.1 the html content of the element you are passing is flatten, meaning that elements you use the plugin with MUST contain only text and not other html childnodes if you don't want undesired effects (this may change in a close future).
