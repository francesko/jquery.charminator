jQuery Charminator

A plugin to achieve interesting text effects. Try it out in the test below or visit the github page for more info. You can download the source here or get the minified version here. Also check a slightly more complex example here

Note that this plugin works better with monospaced font families as all letters have the same size (this avoids text flickering when changing from one letter to another).

DISCLAIMER: As of version 0.0.1 the html content of the element you are passing gets flatten, meaning that elements you use the plugin with MUST contain only text and not other html childnodes if you don't want undesired effects.
